# Support Policy

Thank you for purchasing LegalPage AI. This support policy explains what is included, what is not included, and how to request help.

## Included Support

Support includes help with:

- Installing the project locally
- Understanding the documentation
- Configuring environment variables
- Running the development server
- Running production builds
- Understanding included features
- Basic setup of Stripe keys and price IDs
- Basic setup of LemonSqueezy keys and variant IDs
- Locating files and components
- Understanding demo accounts and demo data
- Reporting reproducible bugs in the original package

## Not Included in Support

Support does not include:

- Custom feature development
- Custom design work
- Rewriting the data layer for a specific database
- Server administration
- Hosting account configuration
- Third-party account creation
- Stripe account approval
- LemonSqueezy account approval
- OpenAI, Gemini, or Anthropic billing issues
- Legal advice
- Writing custom legal documents
- Debugging changes made after heavy customization
- Integrating unsupported payment providers
- Multi-tenant enterprise architecture consulting

## Response Expectations

Support requests are reviewed during business days. Please allow reasonable time for replies, especially when a request requires reproduction or environment review.

## Before Requesting Support

Please check:

1. `README.md`
2. `INSTALLATION.md`
3. `DEPLOYMENT.md`
4. `ADMIN_GUIDE.md`
5. `BILLING_SETUP.md`
6. `STRIPE_SETUP.md`
7. `LEMONSQUEEZY_SETUP.md`
8. `AFFILIATE_SYSTEM.md`
9. `.env.example`

Also verify:

- Node.js version meets the requirement
- Dependencies were installed with `npm install`
- The app builds with `npm run build`
- Required environment variables are present
- AI provider keys are valid
- Stripe/LemonSqueezy webhook secrets match your provider dashboard

## How to Submit a Useful Support Request

Include:

- Purchase code if required by the marketplace
- Your Node.js version
- Your npm version
- Hosting environment
- Exact error message
- Steps to reproduce
- Screenshot or terminal output
- Whether the issue happens before or after customization

## Customization Services

Custom work such as database conversion, extra payment gateways, custom UI themes, custom legal templates, or advanced tenant architecture is outside standard item support. These may be handled as paid customization work if available.

## Legal Disclaimer

LegalPage AI generates AI-assisted drafts and does not provide legal advice. Support does not include legal review, compliance guarantees, or jurisdiction-specific legal recommendations.
