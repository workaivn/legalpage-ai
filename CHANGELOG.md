# Changelog

## 1.0.0 - Marketplace Release

- Added Free, Pro, and Agency subscription plans.
- Added Stripe and LemonSqueezy checkout and webhook routes.
- Added usage quotas, credits, PDF credit costs, and provider usage tracking.
- Added billing history, coupon system, affiliate/referral system, and commission settings.
- Added Agency team workspace and white label mode.
- Added admin monetization metrics for MRR, ARR, active subscriptions, churn, conversion, and revenue by month.
- Added billing, Stripe, LemonSqueezy, and affiliate documentation.
- Added email/password authentication.
- Added user dashboard and admin panel.
- Added document save, edit, duplicate, delete, search, and filter.
- Added generators for GDPR Notice, Disclaimer, Acceptable Use Policy, DPA, AI Usage Policy, Affiliate Disclosure, and Copyright Notice.
- Added OpenAI, Gemini, and Anthropic provider settings.
- Added Markdown, HTML, and PDF exports.
- Added template marketplace with seven categories.
- Added SEO metadata, sitemap, robots, structured data, and social metadata.
- Added demo data and marketplace documentation.
- Added dark mode support and production UI states.
